

                     URL Design


* sign up page

  https://pastehub.net/   (Rails)


* web api

  https://pastehub.net/api/
   => mount point for web api
      http://webapi.local:8000/

  https://pastehub.net/notifier/
   => mount point for notifier api
      http://webapi.local:8001/


* apache settings

    ProxyPass          /api            http://webapi.local:8000/
    ProxyPassReverse   /api            http://webapi.local:8000/
    ProxyPass          /notifier       http://webapi.local:8001/
    ProxyPassReverse   /notifier       http://webapi.local:8001/
    ProxyPass          /               http://webpage.local:3000/
    ProxyPassReverse   /               http://webpage.local:3000/

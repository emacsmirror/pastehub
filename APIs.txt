Realtime db sync sample APIs
=======================

## authentication

 All API method must use this user authentication logic

 1) x-synchrobase-username  : userA
 2) x-synchrobase-date      : 913279782384         (epoc time by seconds)
 3) x-synchrobase-key       : 2987SDFHASDF828SX
 4) x-signature             : 9283740FADB341DA213153

 4) is HMAC-SHA-256 of concat string of 1) and 2) and 3) .
    HMAC-SHA-256 of concat string like "key:value,key:value:key:value"

 A) if x-synchrobase-date and current time differs over 5 minute,
    authentication will fail.

 B) if x-signature calced by local-side and calced by server-site differs,
    authentication will fail.

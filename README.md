PasteHub
=======================


## Concept

Cross-platform clipboard (copy and paste) sync tool.

![image](./doc/client/pastehub_architecture.png)

## Support OS

   type        | OS's clipboard |   Emacs kill-ring  |    vim    |
---------------|----------------|--------------------|-----------|
MacOSX(gem)    |        OK      |         OK         |    OK     |
Windows(gem)   |        OK      |      not yet       |  not yet  |
Linux(gem)     |      not yet   |         OK         |    OK     |

PasteHub
=======================

## Server Sample

1. enviroment veriable

  export JRUBY_HOME=/usr/local

2. config file

/etc/pastehub.conf
{
  "dbPath":        "/var/pastehub/",
  "memcacheHost":  "localhost:11211"
}

3. Create tables on DynamoDB
  1) users
    tablename:  pastehub.org_users
    key:        username 

  2) entries
    tablename:  pastehub.org_entries
    key:        userkey

## Client Sample

1. enviroment variable

  You must get your username and secret key from cloud service
  (pastehub.org or pastehub.net)
  And setup these variables.

  export PASTEHUB_USER="*YOUR USERNAME*"
  export PASTEHUB_SECRET_KEY="*YOUR SECRET KEY*"

2. mkdir for client 

  mkdir ~/.pastehub

3. config file

~/.pastehub.conf
{
  "targetApiHost":       "localhost:8000",
  "targetNotifierHost":  "localhost:8001",
  "localDbPath":         "/tmp/local/"
}
